# To Do App - PERN

To do app using PERN stack.
Can add, edit, delete and  mark as done any to-do listed.

Following freeCodeCamp tutorial on https://www.youtube.com/watch?v=ldYcgPKEZC8&t=4040s

## To run app

## Database

Follow commands on *to_do_db.sql* in *backend/src* to create user, database and tables.

### Backend

On backend/

- `npm install`
- `npm start`

### Frontend

On frontend/

- `npm install`
- `npm start`

Then go to *localhost:9000*
